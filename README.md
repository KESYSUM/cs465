# cs465

Overview

This artifact includes my completed full stack web application. The goal of this project was to build a working application that supports both customer and administrative functionality, while also implementing secure login authentication for admin access. This project demonstrates my ability to connect frontend and backend systems and create a fully functional, secure application.

Architecture

In this project, I used multiple types of frontend development, including Express HTML, JavaScript, and a single-page application (SPA). Express HTML is more traditional and reloads pages when navigating, while the SPA provides a smoother experience by dynamically updating content without reloading the entire page. The SPA felt more modern and efficient, especially for user interaction.

The backend uses a NoSQL MongoDB database because it is flexible and works well with JSON-style data. It allows for easy storage and retrieval of data without needing a strict schema, which made development faster and easier to adjust as the project evolved.

Functionality

JSON is different from JavaScript because it is a data format, not a programming language. It is used to store and transfer data between the frontend and backend. In this project, JSON connects everything by allowing the frontend to send and receive data from the backend API.

During development, I refactored code multiple times to improve efficiency and organization. Breaking code into reusable UI components made the application cleaner and easier to manage. The main benefit of reusable components is that they save time, reduce duplication, and make updates easier across the app.

Testing

Testing involved making sure API endpoints worked correctly using methods like GET, POST, PUT, and DELETE. Each endpoint had to be tested to ensure it returned the correct data and handled requests properly. Adding security, like admin authentication, made testing more complex because it required verifying access control and making sure only authorized users could perform certain actions. Understanding how methods, endpoints, and security work together is important for building a reliable full stack application.

Reflection

This course helped me get closer to my professional goals by giving me real experience building a full stack application from start to finish. I developed skills in frontend and backend development, working with APIs, using databases, and implementing security features. I also improved my ability to organize code, debug issues, and think through the full development process. These skills make me a stronger and more marketable candidate in the computer science field.
